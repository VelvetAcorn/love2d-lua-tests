# LÖVE2D Test Project

A simple LÖVE2D game that displays a rectangle that moves to a random position when you press the spacebar.

## Running the Game

To run this game, use the following command:

```bash
/home/deck/Applications/love-11.5-x86_64.AppImage /home/deck/Documents/GitHub/love2d-lua-tests/zz-ryan-test
```

## Controls

- **Spacebar**: Move the rectangle to a random position

## Files

- `main.lua` - Main game file with rectangle drawing and movement logic
